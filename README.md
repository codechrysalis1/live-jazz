# Live Jazz

Connecting fans with local jazz musicians.

## Getting Started

```
git clone
yarn install
```

### Prerequisites

- node > 8.0.0
- npm
- yarn

### Installing

1. Clone
```
git clone https://github.com/codechrysalis1/live-jazz.git
```

2. Install
```
yarn install
```

## Running the tests

```
yarn test
```

### Coding style tests

```
yarn lint
```

## Deployment

Deployed using Heroku at [https://live-jazz.herokuapp.com/](https://live-jazz.herokuapp.com/)
